## 📋 Sobre o Projeto

Este é meu portfólio pessoal como **Desenvolvedor Front-end**, desenvolvido com foco em design moderno, responsividade e acessibilidade. O site apresenta minha trajetória, tecnologias que utilizo e projetos desenvolvidos.

---

## ✨ Funcionalidades

- **Seção About** — apresentação pessoal com foto e descrição profissional
- **Seção Tecnologias** — exibição das stacks dominadas
- **Seção Projetos** — cards interativos com link para visualização e repositório de cada projeto
- **Design dark com tema espacial** — efeito de partículas/estrelas no fundo
- **Totalmente responsivo**

## 🖥️ Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/johnbalbino/portfolio.git

# Acesse a pasta do projeto
cd portfolio

# Abra o arquivo index.html no seu navegador
# Ou utilize a extensão Live Server no VS Code
```

> Não há dependências externas ou necessidade de instalação — é puro HTML, CSS e JavaScript!
